Escape sequences
\n - Newline
\a - Beep
\b - Backspace
\t - Tab
\\ - Backslash
\' - Single quote
\" - Double quote
\f - Formfeed
\r - Carriage return
\v - Vertical tab
\? - Literal question mark (trigraph related)
\xHH - Hexadecimal numbers
\UHH - Unicode hexadecimal values
\uHH - Unicode hexadecimal values < 10000

Conversion specification character %
char     - %c
int      - %d %4d at least 4 chars wide
float    - %f %4f at least 4 chars wide || %.2f with 2 chars after decimal point.
double   - %f
unsigned - %u
octal	- %o
hex	  - %x
string   - %s
